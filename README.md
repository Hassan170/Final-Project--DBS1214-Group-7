Hotel Booking Cancellation Prediction

This repository contains the dataset and preprocessing work for a data mining project on hotel booking cancellation prediction.

Contents
- Jupyter Notebook with data preprocessing, EDA, and feature selection
- Hotel Booking Demand dataset (source: Kaggle)
- Visual outputs used in the report

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

This repository supports the analysis presented in the final project report
